Exemple d'ús de variables

La definició de variables funciona de la següent manera: Al CSS, s'ha de definir un estil per a :root, on s'aniran definint les variables.
Les variables es defineixen de la següent manera:
--nomVariable:
Després dels dos punts (:) cal posar el valor que tindrà aquesta variable.
Per exemple, per definir una variable que guardi el valor d'un color:
--colorClar: yellow;
